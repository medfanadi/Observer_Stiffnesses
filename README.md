# Nonlinear Observer for Vehicle Dynamics: Joint Front & Rear Cornering Stiffness Estimation

[![MATLAB](https://img.shields.io/badge/MATLAB-R2023b%2B-blue.svg)](https://www.mathworks.com/products/matlab.html)
[![Simulink](https://img.shields.io/badge/Simulink-Enabled-orange.svg)](https://www.mathworks.com/products/simulink.html)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

This repository hosts a high-performance **Nonlinear Observer Architecture** implemented in MATLAB and Simulink. The primary core objective is the real-time joint estimation of vehicle states (lateral velocity, yaw rate) along with time-varying parameters, specifically the **front and rear tire cornering stiffnesses** ($C_{\alpha f}$ and $C_{\alpha r}$).
